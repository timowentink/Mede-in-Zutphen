Componenten

Icons

Icons komen van Phosphor Icons — phosphoricons.com. Laden via CDN of npm pakket @phosphor-icons/web.

Installatie
CDN (in <head>)
<script src="https://unpkg.com/@phosphor-icons/web"></script>
NPM
npm install @phosphor-icons/web

// In je entry file:
import '@phosphor-icons/web';
Gebruik in Blade / HTML
<!-- Regular (default) -->
<i class="ph ph-heart"></i>

<!-- Bold -->
<i class="ph-bold ph-heart"></i>

<!-- Fill -->
<i class="ph-fill ph-heart"></i>

<!-- Grootte via font-size -->
<i class="ph ph-heart" style="font-size: 24px"></i>

<!-- In een button -->
<button class="btn btn-m btn-round btn-blue">
  <i class="ph ph-arrow-right"></i>
  Ontdekken
</button>
